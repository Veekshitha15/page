# Freelancing Landing Page

This project is a static landing page built from a Figma reference design.

## Tech Stack

- HTML5
- CSS3

## Project Structure

- `index.html` - page structure and content
- `styles.css` - all styling and layout rules
- `assets/` - exported images and SVG icons from the Figma design
- `preview.png` - quick visual preview of the landing page
